1. Atributos Básicos

1.1. class y id

class: Es como ponerle una etiqueta a un grupo de cosas. Si tienes varios juguetes, puedes etiquetarlos a todos como "juguetes".
id: Es como el nombre único de una cosa. Si tienes un perro llamado "Firulais", solo hay un Firulais.

Ejemplo:

   
<div class="animal" id="perro1">
<p>Este es Firulais.</p>
</div>
<div class="animal" id="gato1">
<p>Este es Micifuz.</p>
</div>

1.2. style

style: Es como pintar un dibujo. Puedes cambiar el color, el tamaño, y muchas cosas más. Aunque es mejor usar hojas de estilo externas (CSS) para una mejor organización, style es útil para cambios rápidos y sencillos.
