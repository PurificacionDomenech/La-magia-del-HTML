Estilos HTML - CSS

Con CSS podemos cambiar los atributos de los elementos HTML, su color, forma, tamaño,...

   Los CSS se pueden añadir a los documentos HTML de 3 maneras
             -En linea: usando el atriburto style= dentro de los elementos HTML
             -Interno: usando un elemento (style)en la sección (head)
             -Externo: usando un elemento(link) para vinvular a un archivo css externo

    La forma más común de añadir css es mantener los estilos CSS en un archivo externo


div{background-color:hsla(186,100%,50%,0.6);}
pre{background-color:hsl(186,100%,50%);}
p {
border:2px solid white;}
(/style)
(/head)

Archivo Css externo ejemplo
NombreArchivoEstilo.ccs
body{
background-color:red;} h1{....}
(links rel="styleheet" href=NombreArchivoEstilo.css)

Resumen

  • style atributo paraestilo en linea.
  • (style) elemento HTML para definir CSS interno.
  • (link) elemento HTML hace referencia de archivo CSS externo
  • (head) elemento HTML almacena los elementos ( style) y (link)
    • Utilice las propiedades CSS como:
  • color: color texto
  • font-family: fuente del texto
  • font-size: tamaño del texto
  • border: bordes
  • padding:espacio dentro del borde
  • margin: espacio fuera del borde
